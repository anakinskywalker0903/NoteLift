# NoteLift

NoteLift is a Chrome Extension that helps students turn highlighted text from
web pages and PDFs into clean, module-wise revision notes.

## Features
- Capture highlighted text from web pages
- Capture text from PDFs (copy â†’ capture)
- Explicit module-based organization
- Local-only storage (no cloud, no backend)
- Export notes as a `.txt` file
- Safe reset option

## How to Use
1. Set a module name
2. Highlight text (or copy from PDF)
3. Capture into NoteLift
4. Export when ready

## Installation (Developer Mode)
1. Clone this repository
2. Open `chrome://extensions`
3. Enable Developer Mode
4. Click **Load unpacked**
5. Select the NoteLift folder

## Privacy
All data stays locally in your browser.
No tracking, no analytics, no external APIs.
